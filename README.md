# na-learn-vue

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Run development server with json-server
```
json-server db.json
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
